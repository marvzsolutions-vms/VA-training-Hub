# V2.5 communication, branding and referrals

## Added
- Email-style personal notifications for any user, including Zoom invitations and access requests.
- Notification detail view with sender name/email, details, action button and external link.
- News-style clickable announcements with optional uploaded banner image.
- Academy logo upload and immediate shell branding refresh.
- Student and coach profile photo upload plus standard male/female avatars.
- Referral ownership field for students. Coaches see only a “My referral” marker for their own referrals; managers and owners see and assign all coach referrals.
